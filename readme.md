### Code examples for React 2020 
Start here: <https://www.tocode.co.il/bundles/react>

## Prerequisites

* Node JS

## Setup

Run:
 
    git clone 
    cd react-course
    npm install

## Running exercises:

For example, to view example 03, React on Windows, run:
   
    cd 03-react-on-windows/before
    npx webpack-dev-server -d --open
